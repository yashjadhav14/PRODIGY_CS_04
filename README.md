# 🔐 PRODIGY_CS_04 - Simple Keylogger

## 📌 Task-04: Keylogger Implementation

This project is part of my Cybersecurity Internship at Prodigy Infotech.

---

## 📖 Project Overview
In this task, I developed a basic keylogger using Python that records keystrokes and saves them into a file. This project is created strictly for educational purposes to understand how keylogging works in cybersecurity.

---

## ⚙️ How It Works
- The program listens for keyboard inputs
- Each key press is captured
- The pressed keys are stored in a text file (`keylog.txt`)
- The program stops when the ESC key is pressed

---

## ✨ Features
- Logs all keystrokes in real-time
- Saves data into a file
- Handles special keys (Enter, Space, etc.)
- Simple and lightweight implementation

---

- Python
- pynput library
